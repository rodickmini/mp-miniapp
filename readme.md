# 美篇专业版微信小程序

## 命令
``` bash
# 开发模式运行项目
npm run dev

# 生成正式环境代码
npm run build

# 生成开发模式web版本
npm run dev:web

# 清理dist
npm run clean

# 使用eslint修正代码格式
npm run fix
```

## 使用   
`npm run dev` 后会生成 `dist` 文件夹，使用 `微信开发者工具` 选择该文件夹即可，项目具有热更新特性
